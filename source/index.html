What is Litework?

Litework is a simple and light-weight framework for building responsive websites in hand-coded HTML/CSS. Simply replace the HTML content with your own and customise the CSS as neccesary.

Litework was created by Ash Edwards in order to simplify the process of creating new websites and to minimize the time wasted from rewriting the same code repeatedly.
